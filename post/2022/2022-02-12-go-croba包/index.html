cobra包可以实现命令行应用，很多开源工具都在使用，比如：kubernetes、docker、filebeat、Hugo等。官方文档给出了cobra的脚手架使用和使用指南

cobra的用法和常接触的命令用法相同，比如git clone URL、go get -u URL等。cobra分为根命令rootCmd和子命令，可以通过cobra命令添加子命令或者自行添加子命令。flag用来修饰命令的具体行为。

环境准备和例子

首先下载cobra包go install github.com/spf13/cobra/cobra，下载完成后确认cobra在你的本地命令行可以使用，如果不能使用后续可以尝试~/go/bin/cobra，其中~/go是GOPATH。cobra init appname可以直接创建一个cobra项目，根据下面给出例子。

  1. 创建项目目录，并创建go mod项目，
1
2
3

mkdir myapp
cd myapp
go mod init myapp
  1. 初始化 cobra 应用
1
2
3

cd myapp
cobra init myapp    # 如果出现 command not found: cobra， 可以使用 ~/go/bin/cobra init myapp
cobra add serve     # 添加子命令

需要保证项目的目录结构如下，serve.go是子命令对应的文件。

基本的调用顺序：main->rootCmd.Execute()->cmd.execute(flags)->c.Run(c, argWoFlags)，在调用中cmd是rootCmd，c是serveCmd。其中Command结构体的定义如下（删去没用到的变量）

type Command struct {
	Use string
	Short string
	Long string
	Run func(cmd *Command, args []string)  // 匿名函数类型
}

在serve.go中，init函数中添加了子命令，serve变量中定义了Command结构体其中包括Run变量，所以通过serveCmd.Run就能调用子命令的逻辑

package cmd
import (
	"fmt"
	"github.com/spf13/cobra"
)

// serveCmd represents the serve command
var serveCmd = &cobra.Command{
	Use:   "serve",
	Short: "A brief description of your command",
	Long: `A longer description that spans multiple lines and likely contains examples
and usage of using your command. For example:

Cobra is a CLI library for Go that empowers applications.
This application is a tool to generate the needed files
to quickly create a Cobra application.`,
	Run: func(cmd *cobra.Command, args []string) {
		fmt.Println("serve called")
	},
}

func init() {
	rootCmd.AddCommand(serveCmd)
}
